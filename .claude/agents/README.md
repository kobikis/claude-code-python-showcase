# Claude Code Agents - Python Project Specialists

Specialized agents for complex Python development tasks.

## Available Agents

### Architecture & Design
- `architecture-reviewer.md` - Review system architecture and design decisions
- `refactor-planner.md` - Plan large-scale refactoring efforts

### Code Quality
- `code-refactor-master.md` - Execute refactoring with Python best practices
- `python-code-reviewer.md` - Review Python code for quality and style

### Debugging & Fixing
- `auto-error-resolver.md` - Automatically resolve common Python errors
- `api-debugger.md` - Debug FastAPI/Django API issues

### Documentation
- `documentation-architect.md` - Create comprehensive project documentation
- `api-documenter.md` - Generate OpenAPI/Swagger documentation

### Testing
- `test-coverage-improver.md` - Improve pytest test coverage
- `api-tester.md` - Test API endpoints systematically

## Usage

Invoke agents using the Task tool with the appropriate subagent_type, or manually via:

```
/agent architecture-reviewer
```

## When to Use Agents

- **Complex multi-step tasks** that require sustained focus
- **Specialized expertise** in specific domains
- **Repetitive analysis** across many files
- **Deep dives** into specific problems

Agents have access to all tools and can work autonomously on your behalf.
